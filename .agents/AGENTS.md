# Інструкції та промпти проекту Chedryk_Landing

## Технічні вимоги (Technical Specification)
- **Frontend**: React (Vite + React, TypeScript/JavaScript), Tailwind CSS або Vanilla CSS/SCSS.
- **Backend**: PHP (скрипт/API для обробки POST-запитів консультації, відправка в Telegram Bot API / Email).
- **Docker & Radmin VPN**:
  - `docker-compose.yml` з сервісами `frontend` та `backend`.
  - Прив'язка портів до `0.0.0.0` (наприклад `0.0.0.0:80:80`), щоб сайт відкривався через локальну мережу та Radmin VPN за IP-адресою хоста.
- **Анімації**: Скрол-анімації для кроків виконання робіт (Framer Motion, AOS або Intersection Observer).

## Загальна структура та контент сайту (General & Content)
- **Клієнт**: Чедрик Іван.
- **Регіон**: Закарпатська область.
- **Основні послуги**:
  1. Побудова сонячних станцій (28–60 кВт).
  2. Установка панелей на дах.
  3. Установлення систем EcoFlow та акумуляторів.
  4. Проектування та налагодження електромереж в будівлях.
- **Покроковий процес**:
  - Крок 1: Консультація та виїзд на об'єкт.
  - Крок 2: Проектування та підбір обладнання.
  - Крок 3: Доставка та монтаж.
  - Крок 4: Налагодження та запуск.
  - Крок 5: Сервісне обслуговування.
- **Сторінки**: Головна (Landing) та Сторінка Контактів.
