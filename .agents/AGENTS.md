# Інструкції та промпти проекту Chedryk_Landing

## Технічні вимоги (Technical Specification)
- **Frontend**: React (Vite + React, TypeScript/JavaScript), Tailwind CSS (збережена єдина оранжево-бурштинова колірна схема з можливістю перемикання темної та світлої теми).
- **Backend**: PHP (скрипт/API для обробки POST-запитів консультації, відправка в Telegram Bot API / Email).
- **Docker & Radmin VPN**:
  - `docker-compose.yml` з сервісами `frontend` та `backend`.
  - Прив'язка портів до `0.0.0.0` (наприклад `0.0.0.0:80:80`), щоб сайт відкривався через локальну мережу та Radmin VPN за IP-адресою хоста.
- **Анімації**: Скрол-анімації для кроків виконання робіт (Framer Motion, AOS або Intersection Observer).

## Загальна структура та контент сайту (General & Content)
- **Клієнт**: Чедрик Іван.
- **Регіони роботи**: Закарпатська область та Івано-Франківська область (Ужгород, Мукачево, Хуст, Берегове, Івано-Франківськ, Коломия, Калуш, Яремче та всі села).
- **Основні послуги**:
  1. Побудова сонячних станцій (5 кВт – 1 МВт).
  2. Установка панелей на дах.
  3. Установлення систем EcoFlow, гібридів Deye та LiFePO4 акумуляторів.
  4. Проектування та налагодження електромереж у будівлях.
  5. Супровід документації в Обленерго та Зелений Тариф під ключ.
- **Покроковий процес**:
  - Крок 1: Консультація та виїзд на об'єкт.
  - Крок 2: Проектування та дозволи Обленерго.
  - Крок 3: Доставка та монтаж гібридів Deye.
  - Крок 4: Налагодження та запуск.
  - Крок 5: Зелений Тариф та сервісне обслуговування.
- **Сторінки**: Головна (Landing), Калькулятор Вартності СЕС (5 кВт – 1 МВт) та Сторінка Контактів (+ Посилання на TikTok).
