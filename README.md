# Chedryk_Landing

Офіційний веб-сайт / лендінг для послуг зі встановлення сонячних електростанцій, акумуляторних систем та електромонтажу від **Чедрика Івана** (Закарпатська область).

## Структура проекту та Промпти

В цьому репозиторії сформовано два детальних промпти для розробки та розгортання сайту:

1. 🛠️ **[Технічний промпт (PROMPT_TECHNICAL.md)](PROMPT_TECHNICAL.md)** — містить специфікацію по стеку (React, PHP backend), Docker setup, налаштування доступності через Radmin VPN (binding 0.0.0.0), структуру компонентів та форматуванню коду.
2. 📄 **[Загальний промпт (PROMPT_GENERAL.md)](PROMPT_GENERAL.md)** — містить детальний опис структури сайту, контенту, послуг (сонячні станції 28-60 кВт, монтаж панелей, EcoFlow/акумулятори, електромережі), покрокових інструкцій зі скрол-анімаціями, а також інформацію про клієнта та регіон.

## Основні послуги
- ☀️ **Побудова сонячних станцій (28–60 кВт)**
- 🏠 **Установка сонячних панелей на дах**
- 🔋 **Установка систем EcoFlow та акумуляторів**
- ⚡ **Проектування та налагодження електромереж в будівлях**

## Запуск та локальний перегляд
- Проект адаптований під запуск у **Docker** (`docker-compose up -d`).
- Порти прокинуті на `0.0.0.0`, що забезпечує можливість перегляду сайту через емулятор локальної мережі **Radmin VPN**.
